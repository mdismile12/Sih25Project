## 🐛 Troubleshooting

### "404 on API endpoints"
→ Ensure .htaccess file is in /update_after_mentoring_1/ directory

### "Database connection failed"
→ Make sure MySQL is running and database_setup.sql was imported

### "Cannot login"
→ Use credentials: VET001 / demo or GOV001 / demo

### "API test panel blank"
→ Open browser console (F12) and check for errors
→ Make sure API_BASE_URL is correct in the HTML

### "No data in portal"
→ Database may not be imported
→ Check status.php to verify database tables exist
